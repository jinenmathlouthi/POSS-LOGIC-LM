# POSS-LOGIC-LM

This repository contains the datasets, experimental results, and source code used for experiments on **POSS-LOGIC-LM**, a possibilistic beam-search extension of LOGIC-LM for LLM-based logical reasoning.

## 1. Repository objective

The goal of this repository is to compare a standard **LogicLM-style neuro-symbolic pipeline** with our proposed **POSS-LOGIC-LM** approach.

The experiments compare two methods:

1. **LogicLM baseline**  
   LogicLM uses the top-ranked formalization, denoted `F1`, and applies self-refinement when the symbolic solver reports a parsing or execution error.

2. **POSS-LOGIC-LM / Ours**  
   POSS-LOGIC-LM uses multiple beam-search candidates `F1-F5`, their sequence-level log-scores, a probability-to-possibility transformation, and possibilistic aggregation over symbolic solver outputs.

The repository provides:

- beam-search databases generated by language models;
- experimental results for LogicLM and POSS-LOGIC-LM;
- cleaned source code for reproducing the experiments;
- notebooks for analysis and result inspection.
