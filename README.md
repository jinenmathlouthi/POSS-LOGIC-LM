# POSS-LOGIC-LM

This repository contains the beam-search databases used for experiments on possibilistic aggregation for logical reasoning with language models.

## Databases

The repository contains beam databases for three datasets:

- FOLIO
- PrOntoQA
- ProofWriter

and three models:

- Llama3
- Phi3.5-mini
- Qwen

Each example contains five candidate formalizations generated by beam search, together with their sequence-level log scores.
